# プライバシーポリシー（Law-GPT）

本サービス（以下「本アプリ」）は、ユーザーが入力した内容をもとに e-Gov 法令 API を利用し、法令情報を検索・提示する AI アシスタントです。本アプリにおける個人情報の取り扱いについて以下のとおり定めます。

## 1. 取得する情報

本アプリは、以下の情報を一時的に取得する場合があります。

- ユーザーが入力したテキスト
- URL（アクセス先ページの法令抽出・分析のため）
- 添付ファイル（内容確認・テキスト抽出のため）
- アプリ内操作・エラーログ

## 2. 情報の利用目的

取得した情報は、以下の目的のみに利用します。

- 法令データの検索・要約・回答生成のため
- エラー解析および品質向上のため
- 本アプリの改善および機能提供のため

## 3. 第三者提供について

本アプリは、次の場合を除き、取得したデータを第三者に提供しません。

- e-Gov 法令 API へのリクエスト（法令情報取得のため）

いずれの場合も、個人情報を特定する目的で利用することはありません。

## 4. データの保持について

- 本アプリは、ユーザーの入力データを **保存しません**。
- 会話内容は AI モデルの学習に使用しない設定を採用しています。
- 取得した情報は、処理終了後速やかに破棄されます。

## 5. Cookie等の利用

本アプリは Cookie を利用しません。  
GitHub Pages の仕組み上、GitHub 側の分析用 Cookie が使用される場合がありますが、本アプリでは一切関与しません。

## 6. 免責事項

- 本アプリは、e-Gov 法令 API のデータに基づき回答を生成します。  
  法令の正確性・最新性は API に依存します。
- 本アプリの回答内容は法律的助言を目的としたものではありません。  
  解釈や最終判断については、必ず一次情報をご確認ください。
